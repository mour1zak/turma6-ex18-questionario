# 📋 Sistema de Cadastro de Questionários

## Descrição
Aplicação para criação e aplicação de questionários utilizando JSON Server como API REST.

## Funcionalidades
- ✅ CRUD de Perguntas (4 tipos: múltipla escolha, texto curto, texto longo, checkbox)
- ✅ CRUD de Formulários
- ✅ Resposta de Formulários
- ✅ Listagem de Respostas

## Instalação
```bash
npm install
npm start  # Inicia JSON Server na porta 3000
